sjPlot - data visualization
------------------------------------------------------------------------------
Collection of several plotting / table output functions for visualizing data and utility functions.


sjPlot - data visualization
------------------------------------------------------------------------------
```r
devtools::install_github("devel", "sjPlot")
```
